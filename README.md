# IPF Wilks Dots Calculator
This Chrome extension is used to calculate DOTS, Wilks, and IPF points/score. 

It is available on all the official stores. Visit the links below to install it.

1. [Chrome Webstore](https://chromewebstore.google.com/detail/dots-ipf-+-wilks-calculat/lpbgjmidhgjgiplioiekhbndafdcolje)

2. [Mozilla Addons](https://addons.mozilla.org/en-US/firefox/addon/dots-ipf-wilks-calculator/)

3. [Edge Addons](https://microsoftedge.microsoft.com/addons/detail/dots-ipf-wilks-calculat/djfpjiabhfpfejomlbpobbbbdojfjead)
